# Программный макет системы прослушивания гидроакустических сигналов

## Описание
Проект представляет собой программный макет для анализа и прослушивания гидроакустических сигналов с применением цифровой обработки.  

## Функционал
- Генерация тестовых сигналов (гармонические, многотональные).
- Загрузка пользовательских сигналов (WAV, MAT, CSV, TXT).
- Блочная обработка сигналов (FFT, overlap, shift).
- Перенос частотной полосы (shift) с заданным шагом, зависящим от параметров перекрытия.
- Аудиоконтроль: воспроизведение и сохранение результатов в формате WAV.
- Визуализация спектра и временной формы.

## Перенос частотной полосы
В системе реализован механизм переноса спектра сигнала на заданное число частотных отсчётов.  
Шаг переноса определяется параметрами блочной обработки, в частности процентом перекрытия (overlap).  
- При большом перекрытии возможен более точный перенос.  
- При малом перекрытии перенос возможен только крупными дискретными шагами.  

Этот функционал позволяет выделять интересующие частотные диапазоны и проводить дальнейший анализ без искажений.
